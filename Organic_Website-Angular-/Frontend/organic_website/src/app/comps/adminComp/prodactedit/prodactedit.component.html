Product Adding

Product Name is required.
Product Name must contain only letters and spaces, between 3 and 20 characters.
Product Description is required.
Description must be between 10 and 100 characters.
Product Price is required.
Price must be a number up to 5 digits.
Stock Quantity is required.
Stock Quantity must be a number up to 4 digits.
Brand Name is required.
Brand Name must contain only letters and spaces, between 3 and 100 characters.
Product Category is required.
Category ID must be between 1 and 4.
Image is required.
